# TP 2 Angular Project

This is a sample Angular project created for TP 2. The project demonstrates the structure of an Angular application, component creation, and two-way data binding using Angular CLI.

## Project Structure

The project is organized as follows:

```
tp2-angular-project
├── src
│   ├── app
│   │   ├── components
│   │   │   ├── header
│   │   │   ├── user-profile
│   │   │   └── data-binding-demo
│   │   ├── app.component.ts
│   │   ├── app.component.html
│   │   ├── app.component.css
│   │   ├── app.module.ts
│   ├── assets
│   ├── environments
│   ├── index.html
│   ├── main.ts
│   └── styles.css
├── angular.json
├── package.json
├── tsconfig.json
├── tsconfig.app.json
├── tsconfig.spec.json
└── README.md
```

## Components

- **Header Component**: Displays the header of the application.
- **User Profile Component**: Represents user profile information.
- **Data Binding Demo Component**: Demonstrates various data binding techniques in Angular.

## Features

- Two-way data binding is implemented in the User Profile component to allow real-time updates of user information.
- The application is structured to separate components, making it easier to manage and scale.

## Getting Started

To run the project locally, follow these steps:

1. Clone the repository.
2. Navigate to the project directory.
3. Install the dependencies using `npm install`.
4. Start the development server with `ng serve`.
5. Open your browser and go to `http://localhost:4200`.
